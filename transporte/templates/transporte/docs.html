Endpoints Disponibles

GET /api/transporte/clientes/

Lista todos los clientes registrados en el sistema.

POST /api/transporte/clientes/

Crea un nuevo cliente. Campos requeridos: codigo, nombre, tipo_cliente.

GET /api/transporte/despachos/

Lista todos los despachos con su estado actual.

POST /api/transporte/despachos/

Crea un nuevo despacho. Campos requeridos: referencia, ruta, cliente, carga.

Ejemplos de Uso

Crear un despacho:

POST /api/transporte/despachos/
{
    "referencia": "DESP-2024-001",
    "ruta": 1,
    "cliente": 1,
    "carga": 1,
    "fecha_programada": "2024-01-15T10:00:00Z",
    "estado": "PENDIENTE"
}

Estados de Despacho

  • PENDIENTE - Despacho programado
  • EN_RUTA - En proceso de entrega
  • ENTREGADO - Completado exitosamente
  • CANCELADO - Despacho cancelado
